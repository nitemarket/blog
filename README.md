# Blog

A blog demo that is built in Laravel.

#### Features
The sections where demostration includes:
  - User authentication (built-in)
  - MySQL integration (storing user info)
  - MongoDB integration (storing blog info)
  - Quill.js editor
  - Delta parser and renderer
  - Twitter Bootstrap

#### Installations
Environment requirements:
  * Laravel 5.4
  * MongoDB 3.2
  * PHP >= 5.6.4
  * MySQL 5.6

MySQL setup:
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=blog
DB_USERNAME=root
DB_PASSWORD=root
```
MongoDB setup:
```
MONGO_CONNECTION=mongodb
MONGO_HOST=127.0.0.1
MONGO_PORT=27017
MONGO_DATABASE=blog
MONGO_USERNAME=jack
MONGO_PASSWORD=123
```
